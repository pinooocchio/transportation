## Project Structure

- **Model Layer** → Defines data structure and entities.
- **Service Layer** → Handles business logic and operations.
- **Controller Layer** → Manages API requests and responses.
- **Postman** → Used for API testing.

## Flow

Postman → Controller → Service → Model
